## 🎨 Customization

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #0066ff;
    --secondary-color: #00d4ff;
    --dark-bg: #0a0e27;
    --card-bg: #1a1f3a;
    --text-light: #e0e0e0;
    --text-dark: #ffffff;
    --accent: #ff6b6b;
}
```
